# Contributing to Livdon Shopify Theme

Thank you for your interest in contributing to the Livdon Shopify Theme!

## Development Setup

1. Clone the repository
2. Install Shopify CLI: `npm install -g @shopify/cli @shopify/theme`
3. Connect to your development store: `shopify theme dev --store=your-store.myshopify.com`

## Code Style

- Use 2 spaces for indentation
- Follow Liquid best practices
- Keep CSS organized and commented
- Use semantic HTML
- Ensure accessibility standards

## Pull Request Process

1. Create a feature branch
2. Make your changes
3. Test thoroughly on a development store
4. Submit a pull request with a clear description
